# Watch Lab Frontend (Next.js)

Полный фронтенд для сайта-сервиса мастерской по ремонту часов:
- главная страница;
- каталог услуг с фильтрами;
- страница мастеров + детальная карточка;
- многошаговая запись онлайн (5 шагов);
- кабинеты: клиент, мастер, админ;
- страница контактов;
- адаптивная верстка (mobile first).

## Стек
- Next.js (App Router) + React + TypeScript
- Данные, уведомления, авторизация пока реализованы как mock/stub слой.

## Запуск
1. Установить Node.js 20+.
2. Установить зависимости:
   ```bash
   npm install
   ```
3. Запустить проект:
   ```bash
   npm run dev
   ```
4. Открыть: `http://localhost:3000`.

## Где находятся backend-заглушки
- Основные данные и бизнес-логика:
  - `src/lib/stubs/data.ts`
  - `src/lib/stubs/api.ts`
- Авторизация (SMS/email имитация):
  - `src/lib/stubs/auth.ts`

В этих файлах добавлены комментарии `BACKEND STUB LOCATION` с точками интеграции.
