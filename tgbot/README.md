# WatchLab Telegram bot

Бот дублирует уведомления из сайта в Telegram.

## Запуск

```bash
node tgbot/bot.mjs
```

Токен читается из переменной `TELEGRAM_BOT_TOKEN` или из файла `tgbot/.env`.

## Привязка пользователя

1. Пользователь открывает бота в Telegram.
2. Отправляет `/start`.
3. Отправляет `/link email@example.com`, где email совпадает с аккаунтом на сайте.

После привязки все новые и наступившие по времени уведомления из окна уведомлений сайта будут отправляться в Telegram.
